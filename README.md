# THREE.js with Typescript Template

## Install
```bash npm install```

---
## Run
```bash npm run test```

#### Navigate to https://localhost:6969


---
## Build

#### Web
```bash npm run pack```

Navigate to [bundle.js](/webpack/dist/bundle.js)


#### Typescript (Development)
```bash npm build``` 

Navigate to [dist](/dist/)

*It's also setup so you can directly push to npm*
